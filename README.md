# Student Marks Analyzer

 Project Description
This is a simple Python console-based project that:
- Takes student names and marks as input
- Calculates average marks
- Finds the topper
- Lists failed students (marks < 40)

 Features
- Uses dictionary to store student data
- Uses loops and conditions
- Calculates statistics
- Beginner-friendly structured code
